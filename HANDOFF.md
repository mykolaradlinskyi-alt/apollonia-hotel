# АПОЛЛОНІЯ · Готель — Передаточный пакет

> Лендинг с двумя правовыми подстраницами для небольшого B&B-отеля в Полтаве. Сестринский проект к стоматологической клинике «Аполлонія» — одно здание, общая бренд-система. Этот документ — единственный источник правды для всей работы по сайту.

**Язык сайта:** украинский
**Тип:** статический сайт + 2 правовые страницы (rules, refund). Возможно расширение до полноценной CMS-управляемой системы с онлайн-бронированием.
**Срочность:** не срочно, главное — качество и согласованность с клиникой.

## 🔗 Связь с проектом клиники

Отель и клиника физически находятся в одном здании по адресу **вул. О. Оксанченка, 29, Полтава**. Это сестринские бренды:

- **Клиника** — стоматологічна клініка-студія Сергія Радлінського, основана в 1996
- **Готель** — на втором этаже того же здания, открыт в 2009, изначально для пациентов клиники и слушателей курсов, сейчас открыт для всех

Если у разработчика уже есть пакет клиники (`apollonia_mockup.html`, `apollonia_brandbook.html`, `HANDOFF.md` клиники) — большинство решений переиспользуется. **Брендбук клиники применим и здесь** — те же цвета, шрифты, логотип, тональность.

Различия:
- Тон отеля — теплее, более гостеприимный, менее «клинический»
- Логотип на сайте отеля сопровождается пометкой **«готель»** (на сайте клиники — **«стоматологічна клініка-студія»**)
- Никаких CTA-кнопок типа «Записатися» или «Забронювати» — клиент специально это убрал в обоих проектах. Контакт только через телефон.

## 🎨 Бренд-система (краткая шпаргалка)

Идентична клинике. CSS-токены:

```css
:root {
  --saffron: #E8A82C;
  --saffron-deep: #C26A1A;
  --saffron-light: #F5D88B;
  --saffron-glow: #FBE9B8;

  --sage: #C3BD95;
  --sage-deep: #8E8A68;
  --sage-mist: #E4E2CC;

  --cream: #FAF5E8;
  --ivory: #F2EBD8;

  --walnut: #3D2418;
  --charcoal: #1F1611;
  --muted: #8B7E70;
  --line: #E8DFC8;
}
```

**Типографика:**
- Заголовки H1–H4, цитаты — **Cormorant Garamond** (часто italic)
- Body, UI, навигация — **Onest**
- Подключение Google Fonts: см. в `<head>` любого из HTML-файлов

**Логотип:**
- Знак (лист-папоротник с монограммой Я) + словесный знак «АПОЛЛОНІЯ»
- На сайте отеля используется в трёх формах:
  - Шапка: знак + словесный знак + пометка «готель»
  - Футер: словесный знак на ореховом фоне (инверсный)
  - Фавикон: только знак

## 🛠 Рекомендуемый стек

**Не использовать WordPress.** Современный headless-подход:

- **Frontend:** Astro (или Next.js)
- **CMS:** Decap CMS (git-based, бесплатный) для номеров, галереи, контента
- **Хостинг:** Cloudflare Pages или Netlify
- **Домен:** `hotel.apollonia.ua` (упоминается в тексте правил повернення коштів) или отдельный второй уровня
- **Аналитика:** Plausible или Cloudflare Analytics

Объём сайта небольшой (1 главная + 2 правовые + потенциально страницы каждого номера) — статика работает идеально.

### Связь с сайтом клиники

Варианты архитектуры:
1. **Два отдельных сайта** на разных доменах (`apollonia.ua` и `hotel.apollonia.ua`) — рекомендуется. Каждый автономен, общая палитра и шрифты задаются дважды (копия CSS-токенов).
2. **Один монорепозиторий** с двумя сайтами — общие компоненты, общая стилистика, разные роутинги. Сложнее, но удобнее для поддержки.

Выбор — за разработчиком и клиентом.

## 🔧 Что нужно реализовать поверх макета

Макет — это **визуальная спецификация**. В нём НЕТ реальной интерактивности. Что должно быть в production:

### Критическое

1. **Адаптивное мобильное меню** — гамбургер. В макете шапка раскрывается, но nav скрывается на узких экранах.

2. **Smooth scroll по якорям** — добавить `html { scroll-behavior: smooth; }` + offset под высоту sticky-шапки.

3. **Sticky top bar** — фон шапки слегка добавляет тень при скролле (опционально).

4. **Страницы номеров (опционально)** — каждая карточка номера может вести на `/rooms/[slug]` с галереей и подробным описанием. На лендинге сейчас вся ключевая инфо есть, отдельные страницы нужны только если будет много фото и подробностей.

### Среднеприоритетное

5. **Интерактивная карта Google Maps** — сейчас в секции «Як нас знайти» статичные фото. При желании клиента — заменить третий блок (карту) на `<iframe>` Google Maps embed по адресу. Координаты по `вул. О. Оксанченка, 29, Полтава`.

6. **Lightbox для галереи «Атмосфера»** — клик по фото открывает увеличенное. Простой плагин типа PhotoSwipe или GLightbox.

7. **Онлайн-бронирование + оплата** — в «Правилах повернення коштів» упоминается оплата через сервис **WayforPay**. Если клиент захочет полноценный booking flow:
   - Простая форма «Забронювати»: даты, тип номера, имя, телефон
   - Расчёт стоимости (с учётом готельного збора и опционального завтрака)
   - Платёжная форма WayforPay (или иная)
   - Письма-подтверждения
   - В админке — таблица бронирований, занятость номеров
   
   Это отдельный модуль и отдельный объём работы. Не делать без явного запроса клиента.

8. **Календарь занятости** — если будет онлайн-бронирование, показывать доступность дат для каждого типа номера.

### Низкоприоритетное

9. **Английская версия** — для зарубежных гостей. Архитектура CMS должна это допускать.

10. **OpenGraph и Twitter Card** — для красивых превью при шере. Сейчас в `<head>` есть базовые `description` и `theme-color`, но нет OG-картинки. Использовать тот же логотип на ореховом фоне.

11. **Schema.org разметка** — `Hotel` тип + `LodgingBusiness`. Это критично для локального SEO в Google.

## 📊 Контент-схема для CMS

Не очень большая. Главное — карточки номеров и галерея.

### Room (Номери)

```yaml
slug: string*               # "standard", "superior", "lux"
title: string*              # "Стандарт"
type_label: string*         # "Стандарт · 6 номерів"
order: int*                 # 1, 2, 3
is_featured: boolean        # default: false (только Суперіор сейчас true)
short_description: string   # "Усе необхідне для затишного нічлігу"
amenities: list[string]*    # ["Кабельне телебачення", "Wi-Fi", ...]
amenity_bonus: text         # Только для Суперіор/Люкс ("Комплімент — чай, кава, печиво...")
price_single: int*          # 1200 (грн за добу за одного гостя)
price_double: int*          # 1500 (грн за добу за двох)
price_extra_bed: int        # 400 (только для Люкса)
photo_main: image*          # 1200×900 (4:3), главное фото для карточки
photos_gallery: list[image] # Дополнительные фото (для будущей страницы номера)
footnote: string            # "+ готельний збір · сніданок 250 ₴ окремо"
```

### GalleryPhoto (Атмосфера)

```yaml
slug: string*
order: int*
image: image*               # квадратное, минимум 600×600
caption: string             # Опционально для alt-текста
category: enum              # "common" | "rooms" | "bathrooms" | "exterior" — для будущих фильтров
```

### StaticContent (содержимое блоков «Про готель», «Як забронювати», «Як нас знайти»)

```yaml
about:
  paragraphs: list[text]
  facts:
    - name: "Розташування"
      value: "Полтава · біля М-03"
    - name: "Формат"
      value: "B&B · 8 номерів"
    # ... и т.д.
  photo: image                # фото администратора (4:3)

booking:
  phone: string
  messengers: list[string]    # ["Viber", "WhatsApp", "Telegram"]
  info_rows:
    - label: "Адреса"
      value: "вул. О. Оксанченка, 29 · Полтава"
    # ... и т.д.

practical:
  parking:
    photo: image
    caption: "Безкоштовне паркування біля входу"
  entrance:
    photo: image
    caption: "Вхід з боку фасаду"
  route:
    photo: image
    caption: "Поруч з трасою М-03 · Київ — Харків"
    google_maps_url: string
```

### LegalPage (правовые страницы)

```yaml
slug: string*               # "pravyla" | "povernennya"
title: string*              # "Правила та умови проживання"
eyebrow: string             # "— Готель «Аполлонія»"
dek: string                 # подзаголовок
toc: list[{anchor, label}]  # навигация левой колонки
body_html: text*            # markdown или rich-text, рендерится в .content-body
last_updated: text
```

## 🧠 Принципы, которые не нарушать

1. **Никаких CTA-кнопок** типа «Забронювати» или «Записатися». Клиент специально это убрал.
2. **Палитра янтарных тонов** — единственная фирменная. Не разбавлять.
3. **Cormorant Garamond** — заголовочный шрифт. Не заменять.
4. **Тон голоса** — спокойный, гостеприимный, чуть архаичный (литературный украинский).
5. **Связь с клиникой** упоминается естественно («над клінікою», «спочатку — для пацієнтів і лекторів клініки»). Не маскировать факт, что отель в клиническом здании — это часть истории и характера.
6. **«АПОЛЛОНІЯ» в заголовках** — словесным знаком (изображением), не набранным текстом.
